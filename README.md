# visuAlgorithm
Visualiser for the following algorithms:
- A* Pathfinder
- Dijkstra's Pathfinder
- Depth First Search
- Breadth First Search
- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Bogo Sort
- Linear Search
- Binary Search
- Randomised Prim's Maze Generation

## How to use the project
Clone the project and then run the command

`./gradlew run`
